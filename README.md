# 🎓 Student Engagement Detection System

## 📌 Overview
This project detects student engagement in real-time using facial emotion recognition and eye tracking.

## 🚀 Features
- Live webcam engagement detection
- Video file analysis
- Emotion detection using CNN
- Attentive vs Distracted tracking
- Streamlit dashboard
- Session history logging

## 🛠️ Tech Stack
- Python
- OpenCV
- Keras (CNN)
- Streamlit
- NumPy, Pandas

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
